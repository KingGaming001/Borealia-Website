# Borealia Website

Starter landing page project for the Government of Borealia, built with HTML5, CSS, JavaScript, and Bootstrap 5.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5 (CDN)
- Vanilla JavaScript

## Project Structure

- `index.html` - Landing page layout and content
- `assets/css/styles.css` - Custom styling and responsive rules
- `assets/js/main.js` - Basic interaction logic
- `assets/img/` - Place image assets here (currently placeholders in markup)

## Run Locally in VS Code

1. Open `index.html` directly in a browser, or use Live Server.
2. If using Live Server, right-click `index.html` and choose `Open with Live Server`.

## Notes

- Current crest, emblem, and icon blocks are placeholders.
- Replace placeholders with final Borealia art assets in `assets/img/`.

## Live Static Hosting (GitHub Pages)

This project is configured for automatic deployment using GitHub Pages.

### One-time setup

1. Create a GitHub repository and push this project to the `main` branch.
2. In GitHub, open `Settings` > `Pages`.
3. Under `Build and deployment`, set `Source` to `GitHub Actions`.
4. Push any new commit to `main`.

GitHub Actions will deploy the site automatically from this repository.

### Ongoing updates

1. Edit files in this workspace.
2. Commit and push to `main`.
3. Wait for the `Deploy Static Site to GitHub Pages` workflow to finish.
4. Refresh your site URL to see changes live.

### Optional custom domain

1. Add your domain in `Settings` > `Pages`.
2. Update DNS records with your domain provider.
3. Enable HTTPS in GitHub Pages settings after DNS is verified.
